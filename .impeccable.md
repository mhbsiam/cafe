# Design Context — CAFE

Persisted context for design/UX work (used by the impeccable/frontend-design skills).

## Target audience
Immunology and flow-cytometry researchers. Bench scientists analyzing spectral
flow cytometry (SFCM) data exported from FlowJo. Not developers; a no-code tool.

## Use cases
- Upload per-sample CSVs, process into an AnnData object.
- Run dimensionality reduction (UMAP) and Leiden clustering.
- Explore clusters/markers/groups and produce **publication-ready** figures.
- Compare experimental groups (e.g. Aged vs Young) with statistics.

## Brand personality / tone
**Serious lab tool.** Precise, trustworthy, restrained. "Wow" should come from
*capability* (fast, interactive, handles real data) — not visual flash.
Publication integrity is paramount: interactive/animated touches must never
distract from or alter the data, and must never replace the static, downloadable
matplotlib figures researchers put in papers.

## Constraints
- Streamlit app; locked light theme (charts are static light-background images).
- Datasets typically < ~100k cells (Plotly Scattergl is sufficient; no heavy WebGL stack).
- All motion is gated behind `prefers-reduced-motion` (see `CAFE/bin/theme.py`).
- Enhancements are additive: never remove/modify the 300 DPI figure exports or ZIP downloads.

## Notes
- Signature accent color: `#0f5070` (`.streamlit/config.toml` primaryColor). Font: Source Sans 3 / sans-serif.
- Landing-page flourishes (dot-field backdrop, scroll reveals) are confined to the
  welcome page only; analysis pages stay businesslike.
- Full design contract lives in `PRODUCT.md` and `DESIGN.md` at the repo root.
