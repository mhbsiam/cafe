This is the simplest way to run CAFE. It installs CAFE as a global command, so
you can open a terminal **anywhere** and just type:

```bash
cafe
```

The app opens in your web browser automatically. No need to `cd` into a folder
or activate an environment first.

Unlike the Pixi and Conda routes, pipx uses **your own** Python — so you need
**Python 3.12** available first.

:::tip[Don't want to manage Python?]
Use the [Pixi Way](/cafe/installation/pixi_way/) or
[Conda Way](/cafe/installation/conda_way/) instead — they install the right
Python for you automatically.
:::

## 1. Install Python 3.12

The easiest option is **[uv](https://docs.astral.sh/uv/)**, a single download
that also fetches the correct Python for you:

```bash
# macOS / Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# Windows (PowerShell)
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

If you use uv, you can skip pipx entirely and use `uv tool install` in place of
`pipx install` in the commands below.

Alternatively, install **Python 3.12** directly from
[python.org](https://www.python.org/downloads/) (or your package manager) and
continue with pipx.

## 2. Install pipx

```bash
# macOS / Linux
python3 -m pip install --user pipx
python3 -m pipx ensurepath

# Windows (PowerShell)
py -m pip install --user pipx
py -m pipx ensurepath
```

Close and reopen your terminal after `ensurepath` so the `cafe` command is
found.

## 3. Install CAFE from GitHub

CAFE is **not published on PyPI**, so you always install it straight from the
GitHub repository:

```bash
pipx install "git+https://github.com/mhbsiam/cafe"
# uv: uv tool install "git+https://github.com/mhbsiam/cafe"
```

Then launch it from any directory:

```bash
cafe
```

To update later, reinstall from the same URL to pull the newest version:

```bash
pipx install --force "git+https://github.com/mhbsiam/cafe"
# uv: uv tool install --force "git+https://github.com/mhbsiam/cafe"
```

The installed package is named `cafe-app`. To remove it:

```bash
pipx uninstall cafe-app
# uv: uv tool uninstall cafe-app
```

## Install from a local wheel

If you downloaded the repository (or a prebuilt `.whl`), you can install from
your local copy instead.

Build a wheel from the source folder:

```bash
python3 -m pip install build
python3 -m build --wheel
```

This writes a file such as `dist/cafe_app-3.0.0-py3-none-any.whl`. Install it
with:

```bash
pipx install dist/cafe_app-3.0.0-py3-none-any.whl
```

Or install the source folder directly:

```bash
pipx install .
```

## Other commands

Installing CAFE also provides two extra commands:

- `cafe-web` — launches the hosted (web) layout with a 2 GB upload limit.
- `cafe-hpc` — the command-line HPC pipeline (see the HPC docs for options).
