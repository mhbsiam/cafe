This is the simplest way to run CAFE. It installs CAFE as a global command, so
you can open a terminal **anywhere** and just type:

```bash
cafe
```

The app opens in your web browser automatically. No need to `cd` into a folder
or activate an environment first.

## Prerequisites

You need **Python 3.12** and **pipx** installed once:

```bash
# Install pipx (macOS/Linux)
python3 -m pip install --user pipx
python3 -m pipx ensurepath

# Windows (PowerShell)
py -m pip install --user pipx
py -m pipx ensurepath
```

Close and reopen your terminal after `ensurepath` so the `cafe` command is
found.

:::tip[Don't have Python yet?]
Use **[uv](https://docs.astral.sh/uv/)** instead — it's a single download that
installs the right Python for you automatically. After installing uv, replace
`pipx install` with `uv tool install` in the commands below.
:::

## Install from GitHub

```bash
pipx install "git+https://github.com/mhbsiam/cafe"
```

Then launch it from any directory:

```bash
cafe
```

To update to the latest version later:

```bash
pipx reinstall cafe-app
```

And to remove it:

```bash
pipx uninstall cafe-app
```

## Install from a local wheel

If you downloaded the repository (or a prebuilt `.whl`), you can install from
your local copy instead.

Build a wheel from the source folder:

```bash
python3 -m pip install build
python3 -m build --wheel
```

This writes a file such as `dist/cafe_app-3.0.0-py3-none-any.whl`. Install it
with:

```bash
pipx install dist/cafe_app-3.0.0-py3-none-any.whl
```

Or install the source folder directly:

```bash
pipx install .
```

## Other commands

Installing CAFE also provides two extra commands:

- `cafe-web` — launches the hosted (web) layout with a 2 GB upload limit.
- `cafe-hpc` — the command-line HPC pipeline (see the HPC docs for options).
