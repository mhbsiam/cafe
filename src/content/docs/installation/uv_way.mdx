This is the **recommended** way to install and run CAFE. It sets CAFE up as a
global command, so you can open a terminal **anywhere** and just type:

```bash
cafe
```

The app opens in your web browser automatically. No need to `cd` into a folder
or activate an environment first.

We use **[uv](https://docs.astral.sh/uv/)** for this route because it also
installs the correct **Python 3.12** for you — so this works even if you don't
have Python installed.

## 1. Install uv

```bash
# macOS / Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# Windows (PowerShell)
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

Close and reopen your terminal afterwards so the `uv` command is found.

## 2. Install CAFE from GitHub

CAFE is **not published on PyPI**, so you always install it straight from the
GitHub repository. uv fetches the right Python automatically during install:

```bash
uv tool install "git+https://github.com/mhbsiam/cafe"
```

Then launch it from any directory:

```bash
cafe
```

This automatically opens CAFE in your web browser. The initial setup may take
longer on your first launch.

To update later, reinstall from the same URL to pull the newest version:

```bash
uv tool install --force "git+https://github.com/mhbsiam/cafe"
```

The installed package is named `cafe-app`. To remove it:

```bash
uv tool uninstall cafe-app
```

:::note[Prefer pipx?]
If you already use [pipx](https://pipx.pypa.io/) and have Python 3.12 installed,
the same commands work with `pipx install` / `pipx uninstall` in place of
`uv tool install` / `uv tool uninstall`.
:::

## Install from a local wheel

If you downloaded the repository (or a prebuilt `.whl`), you can install from
your local copy instead.

Build a wheel from the source folder:

```bash
uv build --wheel
```

This writes a file such as `dist/cafe_app-3.0.0-py3-none-any.whl`. Install it
with:

```bash
uv tool install dist/cafe_app-3.0.0-py3-none-any.whl
```

Or install the source folder directly:

```bash
uv tool install .
```

## Other commands

Installing CAFE also provides an extra command:

- `cafe-hpc` — the command-line HPC pipeline (see the HPC docs for options).
