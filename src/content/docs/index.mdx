## What CAFE does

<CardGrid stagger>
  <Card title="Upload CSVs" icon="upload">
    Import scaled FlowJo exports with the standard SampleID_Group.csv naming convention.
  </Card>
  <Card title="Clean & Correct" icon="approve-check">
    Run MAD-based QC, optional PCA, and ComBat batch correction in a guided pipeline.
  </Card>
  <Card title="Cluster & Explore" icon="open-book">
    Compute UMAP + Leiden clusters, then pan, zoom, and select cells interactively.
  </Card>
  <Card title="Export Figures" icon="download">
    Download 300 DPI PNG/PDF/SVG plots plus a ZIP of tables and the processed AnnData object.
  </Card>
</CardGrid>
